## 📖 About The Project

**BIRO BOT V7** is a powerful, automated stress-testing tool built with Python and Selenium. It features a high-performance engine specifically designed for sending advanced dynamic payloads and custom V-Cards to test the rendering capabilities of messaging applications.

### ✨ Key Features
- **🔥 Ultra-Crash Payload:** Generates randomized, heavy text patterns and invisible characters.
- **⚡ Burst Mode Execution:** Optimized sending intervals to avoid buffer freezing.
- **🔄 V-Attack Switch:** Alternates between heavy text and V-Card payloads to bypass basic filters.
- **🎨 Hacker UI:** Custom colored ASCII terminal interface for a professional look.
- **🌐 Cross-Platform:** Runs seamlessly on Windows, Kali Linux, and Termux.

## ⚙️ Installation & Setup

Choose your operating system below and execute the commands in your terminal to get started.

### 📱 1. Termux (Android)
Ensure you have a stable internet connection before running these commands:

```bash
# Update and upgrade default packages
pkg update && pkg upgrade -y

# Install Python and Git
pkg install python git -y

# Clone the repository
git clone https://github.com/Kingmarw/Biro-Bot-V7.git

# Navigate to the project directory
cd Biro-Bot-V7

# Install the required Python libraries
pip install -r requirements.txt

# Launch the tool
python main.py
```
### 🐧 2. Kali Linux / Ubuntu
**Open your terminal and run:**

## 🚀 Usage Guide
1.Run the script using python main.py.

2.The bot will automatically launch a Google Chrome window.

3.Scan the WhatsApp Web QR Code from your mobile device if prompted.

4.Once your chats load completely, return to the terminal and press Enter.

5.Enter the target phone number (including the country code, e.g., +201...).

6.Watch the terminal for live execution status and duration metrics.


*Note: The bot automatically saves your session data in a local whatsapp_session folder, so you don't need to scan the QR code every time you open the tool.*
